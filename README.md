# Day-1_setup
Selected TensorFlow as the project tool and verified installation. Defined research objectives and planned the implementation roadmap for the internship project.

The primary focus was to understand the structure and expectations of the 14-day AI & ML internship project. The objective was to select a suitable AI tool that aligns with the project requirements and has strong real-world applications.

After reviewing multiple AI frameworks and platforms, TensorFlow was selected as the core tool for research and implementation. TensorFlow is a powerful open-source machine learning framework widely used for building deep learning models. It supports model development, training, optimization, and deployment, making it suitable for both academic research and industrial applications.

The development environment was prepared by installing TensorFlow and verifying the installation. A simple Python script was executed to check the TensorFlow version and confirm that the setup was functioning correctly.

In addition to setup, clear research objectives were defined for the project. These included understanding TensorFlow fundamentals, studying its architecture and working principles, exploring real-world use cases, and implementing an image classification model as a proof-of-concept.

By the end of Day 1, the project direction was clearly established, the technical environment was configured, and a structured roadmap for the upcoming days was prepared. This laid a strong foundation for the research and implementation phases of the internship project.
